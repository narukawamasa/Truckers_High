### サイトテーマ

### テーマを選んだ理由
- 前職でトラック運転手をしていた時に、初めて荷物を積み下ろしする会社のルールなど、知りたくても知れない情報を、運転手同士で共有できるものがあればと常々思っていたからです
- 連絡手段が、電話での通話ということが、ほとんどだったので、もっと簡単に会社との連絡手段があればいいと思ったからです

- 荷物を積み・卸しに行くとき
